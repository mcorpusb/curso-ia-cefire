# Actividad · Formación Profesional
{: .fs-7 }

Diseña un cómic formativo de 6 viñetas sobre una situación profesional real de tu módulo e ilustra cada escena con herramientas de IA generativa.
{: .fs-5 .fw-300 }

---

## 🎯 Objetivo

Utilizar Copilot o ChatGPT para generar el guion de un cómic formativo de 6 viñetas ambientado en un entorno profesional real y, a continuación, producir las ilustraciones viñeta a viñeta con Copilot Image Creator, ChatGPT o Canva para obtener un recurso visual aplicado y listo para usar en el aula-taller.

---

## 🧠 Situación de aula

Eres docente de un ciclo formativo y quieres mostrar al alumnado cómo actuar correctamente ante una situación profesional concreta: un riesgo laboral, un procedimiento de calidad, un error de comunicación con un cliente, el uso incorrecto de maquinaria… Los cómics son especialmente eficaces en FP porque permiten mostrar una decisión incorrecta, sus consecuencias y la corrección, todo sin señalar a nadie.

Con IA puedes generar en minutos un guion con enfoque práctico y profesional, sin teoría abstracta y con el vocabulario técnico del sector.

---

## ⚙️ Qué vas a hacer ( paso a paso)

1. Genera el **guion completo** del cómic con Copilot o ChatGPT.
2. Crea las **imágenes** viñeta a viñeta con Copilot Image Creator o ChatGPT.
3. **Monta el cómic** en Canva, PowerPoint o una plantilla de 6 paneles.
4. Revisa el **vocabulario técnico**, la precisión de los procedimientos y la adecuación al perfil profesional.
5. Usa el cómic como material de seguridad, ficha de aula o recurso de repaso de buenas prácticas profesionales.

## AJUSTA ESTE PROMPT 💬 Prompt listo para usar

```text
Dame un prompt exhaustivo para crear un cómic sobre:

Contexto: Necesito un cómic de [N] viñetas para explicar una situación profesional sobre [TEMA] en el módulo de [MÓDULO].

Actúa como un diseñador de cómics formativos para alumnado de Formación Profesional.

Tarea: Genera un guion con enfoque práctico y profesional. Para cada viñeta incluye:

- VIÑETA [N]:
  - Escena: [Entorno profesional realista: taller, oficina, cocina, laboratorio, aula técnica…]
  - Personajes: [Rol profesional: técnico/a, cliente, compañero/a, responsable…]
  - Acción: [Situación laboral concreta]
  - Diálogo/Texto: [Máximo 30 palabras]
  - Competencia trabajada: [Habilidad, norma o conocimiento profesional]

Estructura:
1. Situación real de trabajo.
2. Aparición de un problema o riesgo.
3. Decisión incorrecta o duda profesional.
4. Intervención correcta.
5. Resolución segura o eficaz.
6. Conclusión profesional o norma clave.

Restricciones:
- Lenguaje técnico accesible.
- Enfoque práctico y aplicado.
- Evitar teoría abstracta.
- Mostrar buenas prácticas profesionales.
- No incluir empresas reales ni datos personales.
```

## 🖼️ Ejemplo de uso

Ilustración estilo cómic profesional. Un técnico joven en un taller
